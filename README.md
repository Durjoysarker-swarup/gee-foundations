# Google Earth Engine (Core Skills)

## Overview
This repository documents my structured learning journey in Google Earth Engine (GEE).  
The focus is to build strong foundational understanding of remote sensing data processing and spatial analysis.

This is a **learning repository**, not a final project.

---

## 🎯 Learning Goals

In this phase, I focus on understanding:

- Satellite image structure and architecture
- ImageCollections and filtering logic
- Spatial operations (ROI and buffering)
- Cloud masking techniques
- Spectral band understanding
- NDVI computation
- Time-series concepts
